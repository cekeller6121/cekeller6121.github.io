Clement's Beer - My first web app after graduating from The Iron Yard, deployed on Heroku. As a mobile first app, users can search their favorite beers from around the world. Users are also able to search by city for (registered) breweries. Too much to drink? Clement's Beer App has you covered with an Uber deep link located at the bottom of every page.
Technologies: HTML/CSS, JavaScript, Node.JS, Express, BreweryDB's API

Clement's Music Streaming App - My Iron Yard week 4 project, and my first project using an API. Deployed on Surge, this app uses SoundCloud's API, returns search results to the page and streams them using an onscreen media player.
Technologies: HTML/CSS, JavaScript, SoundCloud's API
